## Excel表格数据转成json文件数据

1. 在 `create-excel-to-json` 文件夹中 `yarn install` 安装相关 node 依赖
2. 在 `excel-data` 文件夹中放入待转的Excel文件
3. 在 `index.js` 文件中填写相关配置参数
4. 在 `create-excel-to-json` 文件夹中执行 `yarn start` 进行转换
5. 最后会根据定义的json文件生成位置创建文件夹并写入json数据
